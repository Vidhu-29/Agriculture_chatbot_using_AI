How It Works

It is a simple process to interogate with chatbot

1

User Input

The user interacts through text or voice via the chatbot interface.
Examples:
Text: “What fertilizer should I use for Mahaveer rice?” Voice: User speaks into the mic (speech-to-text converts it to text).

2

Language Detection / Selection

The chatbot detects or lets the user choose their preferred language (e.g.,English, Tamil).
Bilingual support ensures inclusive access for all farmers.

3

Natural Language Understanding (NLU)

The input is passed to the Rasa NLU engine.
It performs:
Intent classification: What the user wants (e.g., get_crop_info)
Entity extraction: Key terms like crop name, location, pest type, etc.

4

Dialogue Management (Core)

Rasa Core uses the recognized intent + context to decide the next action.
It may:
Ask for missing info (e.g., crop name)
Fetch weather, crop, or government scheme data
Trigger a custom action or API call

5

Custom Actions / Data Fetching

For dynamic replies, Rasa runs custom Python actions to:
Query weather APIs
Retrieve crop database info (e.g., sowing time, fertilizer dosage)
Fetch scheme details based on user location

6

Response Generation

Based on the action, the bot generates a natural, friendly response.
Example:
“For Mahaveer rice, apply 60 kg urea per acre during tillering stage.”
